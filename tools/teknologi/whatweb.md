# whatweb
Tool ini digunakan untuk menemukan teknologi yang digunakan di suatu web. Tool ini sudah terinstall otomatis di kali linux

- Link download dan instalasi: https://github.com/urbanadventurer/whatweb

- Cara pemakaian
```sh
whatweb <URL>
```
